# ProyectoFeria

## Descripción del proyecto
Este proyecto consiste en el desarrollo de un sistema de Feria Empresarial, realizado en Java, cuyo objetivo es simular la gestión básica de una feria donde participan empresas, se asignan stands y se permite la interacción de visitantes.

## Herramientas y tecnologias utilizadas
- Java JDK 21
- Apache Maven
- NetBeans IDE

## Instrucciones para ejecutar el proyecto

### Requisitos previos
- Tener instalado Java JDK 21
- Tener instalado NetBeans IDE
- Tener Git (opcional, si se clona el repositorio)

### Pasos para ejecutar

1. Clonar el repositorio:
git clone https://github.com/JuanRomero08/ProyectoFeria.git

2. Abrir el proyecto en NetBeans:
- Abrir NetBeans
- Ir a File → Open Project
- Seleccionar la carpeta ProyectoParqueadero

3. Ejecutar el proyecto:
- Click derecho sobre el proyecto
- Seleccionar Run
